# PROJECTO APP
Risk managing web application.


## Setup & Start
1. **System requirements**
    * Install [Node.js](https://nodejs.org/en/)

2. **Install project dependencies** 
    * ```$ cd Projecto ```
    * ```$ npm i```

3. **Run application**
    * in production mode: ```$ npm start```
    * in dev mode: ```$ npm run dev```
